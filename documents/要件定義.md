# プロジェクト定義：LLM学習・推論お試しセット

## 1. プロジェクト概要
Llama 3.2（3B）をベースとしたLLMのファインチューニング（学習）から、成果物のデプロイ、および複数モデルを切り替えて試せる推論環境を「完全無料」かつ「エンジニアフレンドリー」に構築する。

## 2. コア要件

### A. 学習環境 (Training)
- **基盤**: Google Colab (無料版 T4 GPU)
- **インターフェース**: VS Code の Google Colab 拡張機能を使用
- **ライブラリ**: `Unsloth`（高速化・メモリ節約）を使用
- **成果物**: 学習済みのLoRAアダプタを Hugging Face Hub へ直接アップロード（push_to_hub）する

### B. 推論環境 (Inference)
- **基盤**: Hugging Face Spaces (SDK: Gradio)
- **リソース**: 無料版 CPU Basic
- **機能**: 
    - ブラウザから操作可能なチャットUIを提供
    - **動的モデル切り替え**: プルダウンメニューから、Hugging Face Hub上の異なるLoRAアダプタを選択し、即座にロードして推論を実行できること

### C. 管理・運用
- **コード管理**: GitHub を使用（コード、ノートブック、設定ファイル）
- **成果物管理**: Hugging Face Hub を使用（モデル、データセット、推論デモ）
- **秘匿情報**: トークンや秘密鍵は `.env` または各サービスの Secret 管理機能を使用し、コードに直書きしない

## 3. 技術スタック
- **Base Model**: Llama-3.2-3B-Instruct (4-bit quantized)
- **Language**: Python 3.10+
- **Main Libraries**:
    - `unsloth`: 学習高速化
    - `gradio`: 推論UI
    - `transformers`, `peft`, `trl`: LLM操作・学習
    - `huggingface_hub`: モデル共有
- **Data Format**: JSONL (instruction, input, output)
