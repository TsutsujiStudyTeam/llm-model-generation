# 運用設計書

本ドキュメントは、「LLM学習・推論お試しセット」の環境構築から利用までの手順を定義します。

## 1. 概要

本プロジェクトの運用は、大きく「学習フェーズ」と「推論フェーズ」の2つに分かれます。

- **学習フェーズ**: Google Colab を使用して、独自のデータでLLMをファインチューニングし、成果物（LoRAアダプタ）をHugging Face Hubに保存します。
- **推論フェーズ**: Hugging Face Spaces 上にデプロイされたWeb UIを使い、学習させた複数のモデルを切り替えながら試します。

## 2. 学習手順 (ファインチューニング)

### 2.1. 前提条件

- Google アカウント
- Hugging Face アカウント
- Git がインストールされたローカルPC
- VS Code 及び、以下の拡張機能がインストール済みであること
    - `ms-python.python`
    - `ms-vscode-remote.remote-ssh`
    - `Google.colab-ui` (VS Code用Colab拡張)

### 2.2. 手順

1.  **リポジトリの準備**
    - 本プロジェクトのGitHubリポジトリをローカルPCにクローンします。
    - `cd` コマンドでプロジェクトのルートディレクトリに移動します。

2.  **秘匿情報の設定**
    - Hugging Face にログインし、`Access Tokens` ページ ([https://huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)) から、`write` 権限を持つアクセストークンを生成します。
    - VS CodeでColabノートブックを開く前に、ColabのUIから `Secrets` タブ (`{ }`アイコン) を選択し、`HF_TOKEN` という名前でコピーしたトークンを登録します。

3.  **データセットの準備**
    - `data/dataset.jsonl` を、要件に合わせた独自の学習データに置き換えます。フォーマットは `アプリケーション詳細設計書` を参照してください。

4.  **パラメータの編集**
    - `training/params.yaml` を開き、`hf_lora_repo` を自身のHugging Faceユーザー名とリポジトリ名に書き換えます。
    - 必要に応じて、その他の学習ハイパーパラメータを調整します。

5.  **学習の実行**
    - VS Code のコマンドパレット (`Ctrl+Shift+P`) から `Colab: Open Notebook in VS Code` を選択し、`training/finetune.ipynb` を開きます。
    - Colabインスタンスへの接続を確認し、ノートブックのセルを上から順番に実行します。
    - 学習が完了すると、`model.push_to_hub` セルによって、指定したHugging FaceリポジトリにLoRAアダプタが自動でアップロードされます。

6.  **成果物の確認**
    - ブラウザで、`params.yaml` に指定した `hf_lora_repo` のURLにアクセスし、ファイル (`adapter_model.safetensors` 等) がアップロードされていることを確認します。

## 3. 推論環境の構築と利用

### 3.1. デプロイ手順

1.  **Hugging Face Space の作成**
    - Hugging Face にログインし、`New Space` をクリックします。
    - `Space name` を入力し、`License` を選択します。
    - `Space SDK` として `Gradio` を選択します。
    - `Space hardware` は `CPU basic` (無料) を選択します。
    - `Create Space` をクリックして、リポジトリを作成します。

2.  **コードのアップロード**
    - 作成されたSpaceリポジトリのURLをコピーし、ローカルにクローンします。
    - プロジェクト内の `inference/` ディレクトリの中身 (`app.py`, `requirements.txt`) を、クローンしたSpaceリポジトリのルートにコピーします。

3.  **秘匿情報の設定**
    - Space の `Settings` タブに移動し、`Secrets` をクリックします。
    - `New secret` をクリックし、`Name`に `HF_TOKEN`、`Secret value` に学習時と同じ `write` 権限のHugging Faceトークンを入力し、保存します。
        - *Note: `read`権限でも動作しますが、将来的な機能拡張のために`write`を推奨します。*

4.  **デプロイ**
    - Spaceリポジトリにコピーしたファイルを `git add`, `git commit`, `git push` します。
    - Pushが完了すると、Hugging Face Spaces が自動でビルドとデプロイを開始します。`Logs` タブで進捗を確認できます。

### 3.2. 利用手順

1.  **アプリケーションへのアクセス**
    - デプロイが成功すると、`App` タブにGradioアプリケーションが表示されます。
    - URLを共有することで、誰でもアクセスできます。

2.  **モデルの切り替えと推論**
    - 画面上部のプルダウンメニューから、試したいLoRAアダプタを選択します。
    - モデルのロードが完了すると、"モデルを切り替えました" というメッセージが表示されます。
    - 下部のチャットボックスにテキストを入力し、`Enter`キーを押すか送信ボタンをクリックします。
    - モデルからの応答がチャットウィンドウに表示されます。
